# Sensor Summary

## NODE_A

| Sensor | 기능 |
| --- | --- |
| S1 | 수동 공급 시퀀스 시작 입력입니다. 병 투입이 감지될 때까지 대기하는 센서입니다. |
| S2 | 수동 공급 2단계 입력. 병을 놓은 뒤 `S2`가 들어오면 리프트를 중간 위치로 보내고 다시 그립합니다. |
| S3 | 후면 병 공급 자동 트리거 입력입니다. 유휴 상태에서 감지되면 후면 공급 시퀀스를 시작합니다. |
| S4 | 전면 병 공급 자동 트리거 입력입니다. 유휴 상태에서 감지되면 전면 공급 시퀀스를 시작합니다. |
| S5 | 병잔량(%) 확인 |
| S6_conv_bottle_front | 전면 컨베이어 센서 이름으로 정의되어 있지만 현재 동작에서는 직접 쓰이지 않습니다. |
| S7_conv_bottle_rear | 병 배출 컨베이어의 핵심 센서입니다. 병 도착을 감지한 뒤 짧게 더 구동하고 정지해 한 병 배출을 구현합니다. |
| S8_lift_hi | 리프트 상단 위치 확인 센서입니다. |
| S9_lift_mid | 리프트 중간 위치 확인 센서입니다. |
| S10_lift_lo | 리프트 하단 위치 확인 센서입니다. |
| S11_front_bottle_grip | 전면 그리퍼 그립 완료 확인 센서입니다. |
| S12_front_bottle_release | 전면 그리퍼 오픈 완료 확인 센서입니다. |
| S13_rear_bottle_grip | 후면 그리퍼 그립 완료 확인 센서입니다. |
| S14_rear_bottle_release | 후면 그리퍼 오픈 완료 확인 센서입니다. |
| S15 | 예비용 |
| S16 | 서보 위치 완료 확인 센서입니다. 홈 완료와 이동 완료 확인에 사용합니다. |

## NODE_C

| Sensor | 기능 |
| --- | --- |
| S1 | 병 전후 이송축의 후퇴 끝단 센서입니다. 병을 뒤로 빼서 원점이나 대기 위치로 보낼 때 정지 조건입니다. |
| S2 | 병 전후 이송축의 전진 끝단 센서입니다. 병을 앞으로 밀어 넣을 때 정지 조건입니다. |
| S3 | 병 승강축 상승 끝단 센서입니다. 병을 위로 올릴 때 정지 조건입니다. |
| S4 | 병 승강축 하강 끝단 센서입니다. 병을 아래로 내릴 때 정지 조건입니다. |
| S5 | 캡 상하축 상승 끝단 센서입니다. 캡 축을 위로 복귀시킬 때 정지 조건입니다. |
| S6 | 캡 상하축 하강 끝단 센서입니다. 캡 축을 작업 위치로 내릴 때 정지 조건입니다. |
| S7 | 병 좌우 이송축의 좌측 홈 센서입니다. 충전 위치 복귀와 초기 정렬에서 기준점으로 사용됩니다. |
| S8 | 병 좌우 이송축의 반대편 끝단 센서입니다. 현재 메인 시퀀스에서는 직접 쓰이지 않고 보조 테스트에서만 언급됩니다. |
| S9_left_bottle | 좌측 배출 컨베이어 도착 센서입니다. 병이 좌측 배출 위치에 도착했는지 확인합니다. |
| S10_right_bottle | 우측 배출 컨베이어 도착 센서입니다. 병이 우측 배출 위치에 도착했는지 확인합니다. |
| S11_bottle_grip | 병 그리퍼 그립 완료 확인 센서입니다. |
| S12_bottle_release | 병 그리퍼 릴리즈 완료 확인 센서입니다. |
| S13 | 예비용 |
| S14 | 예비용 |
| S15 | 예비용 |
| S16 | 예비용 |
